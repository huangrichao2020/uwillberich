# uwillberich 技能深度学习总结

_完成时间：2026-03-22_

## 📌 核心认知

### 1. 技能的本质

**uwillberich** 不是一个机械的市场分析工具，而是一个**决策导向的 A 股次日策略制定系统**。

核心目标：将今日盘面和隔夜消息转化为**可执行的次日开盘计划**。

### 2. 核心哲学

> "在判断一个事件是'好'还是'坏'之前，先问：
> - 这个事件能否破圈，超越小众交易圈？
> - 能否吸引大资金，而不只是散户讨论？
> - 能否映射到 A 股可识别的板块、产业链或二阶受益者？"

### 3. 注意力 - 流动性链条

```
事件关注度 → 大资金聚焦 → 板块激活 → 散户参与 → 流动性扩张 → 派发
```

**关键判断**：事件当前处于链条的哪个位置？是在加速还是在派发？

## 🎯 市场状态分类器

### 1. 主线市场 (Mainline Market)

**特征**：
- 国家级或全球性催化剂
- 政策或地缘事件，公众关注度高
- 清晰的板块共振，不止一两只股票
- 多个流动性龙头共同上涨
- 广度强或广度显著改善

**高确信度确认**：
- 同一板块多只涨停
- 多个龙头有真实换手且反复确认
- 券商或广泛风险代理确认情绪改善

**默认应对**：
- 聚焦核心板块
- 偏好龙头和流动性强的二线标的
- 允许比区间条件更积极的跟进

### 2. 独立龙头市场 (Independent-Leader Market)

**特征**：
- 只有 1-3 只股票脱离大盘
- 板块本身疲软或混杂
- 催化剂是公司层面的：订单、业绩、重组、资产注入、价格冲击、季节性主题
- 流动性集中在少数标的而非扩散

**默认应对**：
- 交易特定龙头或非常窄的产业链
- 除非跟进扩散，否则不要外推完整板块逻辑
- 保持比真正主线市场更短的周期

### 3. 区间/防御市场 (Range Or Defensive Market)

**特征**：
- 没有令人信服的主线
- 没有持久的独立龙头
- 广度差或不一致
- 强度集中在石油、煤炭、银行、电信、公用事业等低贝塔避险板块

**默认应对**：
- 缩短持仓周期
- 避免盲目追涨
- 偏好观察、选择性均值回归或防御性倾向

## 📊 核心交易原则

1. **相对强度优于盲目抄底**
   - 弱势市场中，最强的存活龙头通常是比最大跌幅者更好的修复候选

2. **区分修复与防御**
   - 如果强度只集中在石油、煤炭、银行、电信、公用事业，市场整体不健康

3. **广度确认，指数本身不确认**
   - 绿色指数但中小盘广度弱通常是低质量的

4. **政策时机很重要**
   - 在 LPR 日期，09:00 的发布可以覆盖纯粹的外部风险设置

5. **板块逻辑必须可理解**
   - 如果散户能快速理解因果链，突破概率更高

6. **偏好二阶受益者而非拥挤的一阶直觉**
   - 在地缘政治或政策冲击中，明显的交易往往太拥挤。更好的交易可能在逻辑链更深一层

7. **持仓周期与催化剂持续时间匹配**
   - 一日游、数周政策周期、长期价值修复不应同等对待

8. **派发与点火同样重要**
   - 如果大龙头停滞而小盘股连续轮动，板块可能正在进入退出流动性而非新扩张

## ⏰ 交易时间门

### 09:00 - 政策门
- 检查 LPR（每月）
- 立即重新排序政策敏感板块

### 09:20-09:25 - 集合竞价
- 检查哪个组领先：
  - 成长修复
  - 政策贝塔
  - 防御集中
  - 孤立独立挤压

### 09:30-10:00 - 开盘窗口
- 确认指数是否守住关键位
- 确认龙头是否收复昨收
- 决定市场足够宽（主线）还是足够窄（独立龙头）

### 10:00-10:30 - 广度窗口
- 拒绝广度差的反弹
- 如果只有少数股票强势，从主线降级为独立或防御

### 14:00-14:30 - 下午确认
- 区分持续修复与技术反弹
- 观察龙头停滞 + 小盘轮动（派发信号）

## 🧠 核心洞察

### 1. 这不是一个长期投资组合系统
uwillberich 专注于**次日战术决策**，而非长期价值积累。LONG 想法应是次要的。

### 2. 市场分类优先于股票推荐
在讨论股票想法之前，必须先将市场分类为：主线 / 独立龙头 / 区间防御。

### 3. 二阶思维是关键
- 明显的一阶交易往往太拥挤
- 更好的交易可能在逻辑链更深一层
- 例如：地缘冲击 → 不直接交易石油，而是交易更差异化的二阶受益者

### 4. 广度是真理
- 指数可以欺骗，但广度不会
- 真正的修复需要广度扩散
- 龙头停滞 + 小盘轮动 = 派发信号

### 5. 时间门是纪律
- 09:00 政策发布可以覆盖隔夜风险设置
- 09:25 集合竞价揭示真实意图
- 09:30-10:00 是确认窗口，不是预测窗口
