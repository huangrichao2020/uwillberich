## ⏰ 时间门总览

```
08:30-08:55  盘前准备     → 隔夜信息扫描，初步分类
09:00        政策门       → LPR 发布，政策重排序
09:20-09:25  集合竞价     → 真实意图揭示
09:30-10:00  开盘窗口     → 方向确认
10:00-10:30  广度窗口     → 真伪验证
14:00-14:30  午后确认     → 持续 or 派发
```

**核心原则**：
> 时间门是硬约束，不是建议。
> 每个时间窗口有明确的检查项和决策规则。
> 错过时间门 = 失去决策优势。

## 🌅 08:30-08:55 盘前准备

### 目标
构建隔夜冲击地图，决定起始假设。

### 检查清单
```
□ 地缘政治：中东、东欧、台海有无升级？
□ 原油价格：布伦特涨跌幅度？
□ 美股方向：纳指、英伟达、特斯拉表现？
□ 汇率：美元/人民币走势？
□ 政策：国务院/发改委/工信部/证监会最新消息？
□ LPR：今日是否发布日 (每月 20 日)？
```

### 决策问题
```
1. 这个事件能否破圈？(超越小众交易圈)
2. 能否吸引大资金？(不只是散户讨论)
3. 能否映射到 A 股板块/产业链？
```

### 起始假设分类
```
□ 主线市场 (mainline)
  条件：国家级催化剂 + 外围稳定 + 政策友好

□ 独立龙头 (independent)
  条件：公司层面消息 + 无板块共振

□ 修复窗口 (repair window)
  条件：外围稳定 + 无重大利空

□ 中性 (neutral)
  条件：信息混杂，方向不明

□ 防御 (defensive)
  条件：外围利空 + 地缘升级 + 商品 spike
```

### 输出模板
```markdown
## 盘前判断 (08:55)
- 起始假设：[主线/独立/修复/中性/防御]
- 隔夜冲击：[地缘/原油/美股/汇率]
- 政策预期：[LPR/部委消息]
- 观察板块：[≤3 个]
- 关键股票：[龙头 3-5 只]
```

## 🚪 09:00 政策门

### 目标
检查 LPR 发布，立即重排序政策敏感板块。

### 检查项 (仅限 LPR 发布日)
```
□ 1Y LPR：[前值]% → [新值]%
□ 5Y LPR：[前值]% → [新值]%
□ 是否符合预期？[是/否]
□ 超预期方向？[宽松/紧缩]
```

### 重排序规则
```
IF 5Y LPR 降息 ≥10bp →
  升级：地产链、家电、建材、券商
  仓位：+10-20%

IF 1Y + 5Y 联合降息 →
  升级：全市场风险偏好
  仓位：+20-30%

IF LPR 不变 →
  不假设政策贝塔会自行引领
  维持原判断

IF LPR 超预期紧缩 →
  降级：地产链、周期
  仓位：-10-20%
```

### 板块敏感度
| 板块 | 5Y LPR 敏感 | 反应时间 | 持续性 |
|------|-----------|---------|-------|
| 地产开发 | ⭐⭐⭐⭐⭐ | 即时 | 1-3 日 |
| 家电 | ⭐⭐⭐⭐ | 即时 | 1-2 日 |
| 建材 | ⭐⭐⭐⭐ | 即时 | 1-2 日 |
| 券商 | ⭐⭐⭐ | 5-10 分钟 | 半日 |
| 银行 | ⭐⭐ | 滞后 | 短暂 |

### 输出模板
```markdown
## 09:00 政策检查
- LPR 发布：[是/否]
- 1Y/5Y：[x% → y%]
- 超预期：[宽松/紧缩/符合]
- 板块重排序：[升级 X/降级 Y]
- 新判断：[维持/调整]
```

## 📢 09:20-09:25 集合竞价

### 目标
识别真实意图，哪个组领先。

### 检查清单
```
□ 哪个组集体高开？
  - 成长修复 (科技、半导体、AI)
  - 政策贝塔 (券商、地产链)
  - 防御集中 (油气、煤炭、银行、电信)
  - 孤立挤压 (1-3 只股票)

□ 龙头股竞价强度？
  - 高开幅度 >2%？
  - 竞价量比 >3？
  - 未匹配买盘充足？

□ 券商/多元金融/银行给出什么信号？
  - 券商强 → 情绪改善
  - 银行强 → 防御信号
  - 分化 → 方向不明

□ 涨停家数预期？
  - >20 家 → 情绪高涨
  - 10-20 家 → 中性
  - <10 家 → 情绪低迷
```

### 竞价强度公式
```
竞价强度 = (高开幅度 × 2) + (竞价量比) + (未匹配买盘倍数)

强势竞价：>8 分
中性竞价：4-8 分
弱势竞价：<4 分
```

### 分类规则
```
IF 成长龙头集体高开 + 券商强 → 成长修复情景
IF 防御板块主导 + 银行强 → 防御集中情景
IF 仅 1-3 只股票高开 → 独立龙头情景
IF 分化严重 + 无明确主线 → 中性观望
```

### 输出模板
```markdown
## 09:25 集合竞价
- 领先组：[成长/防御/独立/分化]
- 龙头竞价：[强/中/弱]
- 券商/银行：[券商强/银行强/分化]
- 涨停预期：[X 家]
- 分类：[成长修复/防御集中/独立龙头/中性]
```

## 🚀 09:30-10:00 开盘窗口

### 目标
确认方向，决定是否主线语言。

### 检查清单 (分时间片)

#### 09:30-09:35 第一波冲击
```
□ 指数开盘位置？[高开/低开/平开]
□ 龙头股第一笔成交？[主动买/主动卖]
□ 板块集体动作？[一致/分化]
```

#### 09:35-09:45 第一轮回撤检验
```
□ 高开后是否回补缺口？
□ 回撤时承接力如何？[强/弱]
□ 龙头是否守住涨幅？[是/否]
```

#### 09:45-10:00 方向确认
```
□ 指数是否收复开盘位？
□ 龙头是否再创新高？
□ 板块跟风是否响应？[3+ 只涨 5%+]
□ 主力流向是否持续？[非脉冲式]
```

### 真强 vs 假强判断
```
真强板块特征：
✓ 高开不回补或快速回补缺口
✓ 龙头封板坚决 (10:00 前)
✓ 板块内跟风股响应 (3+ 只涨 5%+)
✓ 主力持续流入 (非脉冲式)

假强板块特征：
✗ 高开后一路走低
✗ 龙头封板反复打开
✗ 板块内部分化严重
✗ 主力流入后迅速流出
```

### 分类决策
```
IF 指数守住 + 龙头收复 + 广度>50% → 主线市场
IF 指数守住 + 龙头收复 + 广度<30% → 独立龙头
IF 指数跌破 + 龙头水下 → 防御市场
IF 仅防御绿 + 成长弱 → 防御集中
```

### 输出模板
```markdown
## 09:30-10:00 开盘确认
- 指数位置：[守住/跌破]
- 龙头行为：[收复昨收/水下]
- 广度：[上涨 X/Y 家，Z%]
- 主力流向：[持续流入/脉冲/流出]
- 分类：[主线/独立龙头/防御]
- 仓位：[X%]
```

## 📊 10:00-10:30 广度窗口

### 目标
验证真伪，拒绝假反弹。

### 检查清单
```
□ 上涨家数 / 总家数 = ?
□ 涨停家数 / 跌停家数 = ?
□ 修复是否扩散到前 2-3 只龙头之外？
□ 大龙头是否继续推进还是停滞？
□ 小盘股轮动是否伴随大龙头停滞？
```

### 广度阈值与应对
| 上涨家数 | 占比 | 状态 | 仓位 | 应对 |
|---------|------|------|------|------|
| >3500 | >75% | 极强 | 80-90% | 积极跟进 |
| 2500-3500 | 55-75% | 强 | 60-80% | 选择性 |
| 1500-2500 | 33-55% | 中性 | 40-60% | 聚焦龙头 |
| 500-1500 | 11-33% | 弱 | 20-40% | 防御为主 |
| <500 | <11% | 极弱 | 0-20% | 空仓观望 |

### 派发信号 (警惕！)
```
⚠️ 大龙头停滞 (不再创新高)
⚠️ 小盘股轮动涨停
⚠️ 板块内后排补涨加速
⚠️ 主力流入放缓或转流出
⚠️ 广度收窄但 headline 兴奋度高

→ 这是退出流动性，不是新扩张
→ 减仓信号
```

### 降级规则
```
IF 广度<30% → 从主线降级为独立龙头
IF 广度<15% → 从独立龙头降级为防御
IF 小盘轮动 + 龙头停滞 → 派发，减仓
```

### 输出模板
```markdown
## 10:00-10:30 广度验证
- 上涨家数：[X/Y 家，Z%]
- 涨停/跌停：[X/Y 家]
- 扩散情况：[扩散/停滞/派发]
- 仓位调整：[维持/+X%/-X%]
- 新分类：[主线/独立/防御]
```

## 🌞 14:00-14:30 午后确认

### 目标
区分持续修复与技术反弹，决定去留。

### 检查清单
```
□ 早盘强势板块是否维持？
□ 是否有新板块异动？
□ 龙头是否炸板或涨幅收窄？
□ 尾盘资金流向？[流入/流出]
□ 有无突发消息？
```

### 持续 vs 派发判断
```
持续修复特征：
✓ 早盘强势板块维持涨幅
✓ 龙头封板坚决或继续新高
✓ 资金流向稳定或改善
✓ 无突发利空

技术反弹特征：
✗ 早盘强势板块涨幅收窄
✗ 龙头炸板或回落
✗ 尾盘资金流出加速
✗ 新板块异动但无持续性
```

### 决策规则
```
IF 维持强势 + 资金稳定 → 持仓过夜
IF 涨幅收窄 + 资金流出 → 减仓
IF 龙头炸板 + 尾盘跳水 → 清仓
IF 新板块异动 → 观察，不追
```

### 输出模板
```markdown
## 14:00-14:30 午后确认
- 早盘强势：[维持/收窄/消失]
- 龙头状态：[封板/炸板/回落]
- 资金流向：[稳定/流出/跳水]
- 决策：[持仓/减仓/清仓]
- 明日预判：[持续/调整/反转]
```

## ⚠️ 时间门纪律

### 硬约束
```
1. 10:00 前广度未改善 → 不加仓
2. 14:00 后龙头炸板 → 减仓
3. 广度 < 30% → 防御为主
4. LPR 日 09:00 必须检查 → 可覆盖隔夜判断
5. 集合竞价必须记录 → 揭示真实意图
```

### 常见错误
```
✗ 错过 09:25 竞价 → 失去真实意图
✗ 忽视 10:00 广度 → 陷入假反弹
✗ 14:00 不确认 → 过夜风险
✗ LPR 日不看 09:00 → 错过政策分支
✗ 时间门外交易 → 凭直觉而非纪律
```

### 纪律检查
```
每日收盘后自检：
□ 08:55 前完成盘前判断？
□ 09:00 检查 LPR (如适用)？
□ 09:25 记录竞价强度？
□ 10:00 验证广度？
□ 14:00 确认午后状态？
□ 所有决策基于时间门？

连续 5 日满分 → 纪律合格
任何一日缺失 → 重新学习
```
