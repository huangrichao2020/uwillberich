## 🏗️ 三层分析框架

### 第一层：外部冲击层 (External Shock Layer)

**检查清单**：
```
□ 石油和天然气价格冲击
□ 地缘政治事件
□ 美国利率和央行信息
□ 美股和港股指数方向
```

**解释规则**：
```
→ 风险-off 外部冲击通常先伤害高贝塔 A 股成长
→ 如果隔夜冲击稳定，最强的国内成长组往往先修复
→ 战争冲击下，测试直接和二阶受益者
```

## 📊 板块强度评估公式

**板块强度评分**：
```
板块强度 = (涨幅均值 × 0.3) + (主力净流入 × 0.3) + (涨停家数 × 0.2) + (上涨占比 × 0.2)

评分标准：
- 涨幅均值：板块成分股平均涨幅 (%)
- 主力净流入：主力净流入金额 (亿)
- 涨停家数：板块内涨停股票数量
- 上涨占比：板块内上涨股票数 / 总股票数

强度等级：
⭐⭐⭐⭐⭐ > 8 分：极强，主线候选
⭐⭐⭐⭐   6-8 分：强，重点关注
⭐⭐⭐    4-6 分：中性，观察
⭐⭐     2-4 分：弱，回避
⭐      < 2 分：极弱，做空候选
```

## 🔄 产业链传导分析

**传导逻辑模板**：
```
上游 (原材料/芯片)
   ↓
中游 (制造/模块)
   ↓
下游 (整机/系统)
   ↓
终端 (应用/服务)
```

**传导状态判断**：
```
✓ 顺畅传导：上中下游同步上涨，涨幅递减
✗ 阻滞传导：仅单一环节上涨，其他环节无响应
⚠ 过度传导：下游涨幅 > 上游 (不可持续)
```

**CPO 产业链示例**：
```
上游：光芯片 (源杰科技、长光华芯)
   ↓ 涨幅 +3%
中游：光模块 (中际旭创、新易盛、天孚通信)
   ↓ 涨幅 +7% (弹性最大)
下游：AI 服务器 (浪潮信息、工业富联)
   ↓ 涨幅 +2%
终端：云计算 (三大运营商)
   ↓ 涨幅 +1%

结论：中游弹性最大，上游有补涨机会
```

## ⚠️ 降级与派发信号

**降级板块/龙头的条件** (满足 3 条以上)：

```
□ 大龙头连续多日停滞且成交量萎缩
□ 同一板块小盘股连续轮动涨停
□ 小盘股暴涨而板块龙头停止创新高
□ 广度收窄但 headline 兴奋度高
□ 主力流入放缓或转流出
□ 板块内部分化加剧 (龙头涨 10%，后排跌 5%)
□ 利好消息出台但板块不涨反跌
```

**解读**：
> 这通常是派发，不是健康延续。
> 不要混淆"板块还有股票在涨"与"板块还在积累期"。

## 📋 快速决策规则

### 09:00 政策门
```
IF LPR 降息 (5Y) → 升级地产链、家电、建材、券商
```

```
IF 防御板块主导 → 分类为防御

IF 券商/多元金融强 → 情绪改善信号
```

```
IF 指数守住关键位 + 龙头收复昨收 → 真强

IF 仅防御绿 → 防御集中
```

```
IF 尾盘流出加速 → 次日谨慎
```

## 🧠 核心洞察内化

### 1. 广度是真理
> 指数可以欺骗，但广度不会。
> 真正的修复需要广度扩散。

### 2. 主力不会骗人
> 涨 5% 但主力流出 = 散户推动 (不可持续)
> 涨 2% 但主力大幅流入 = 机构建仓 (可持续)

### 3. 相对强度 > 盲目抄底
> 弱势市场中，最强的存活龙头
> 通常是比最大跌幅者更好的修复候选。

### 4. 修复 ≠ 防御
> 如果强度只集中在石油、煤炭、银行、电信、公用事业，
> 市场整体不健康。

### 5. 二阶受益 > 一阶直觉
> 明显的交易往往太拥挤。
> 更好的交易在逻辑链更深一层。

### 6. 派发与点火同样重要
> 如果大龙头停滞而小盘股连续轮动，
> 板块可能正在进入退出流动性而非新扩张。

### 7. 时间门是纪律
> 09:00 政策发布可以覆盖隔夜风险设置
> 09:25 集合竞价揭示真实意图
> 10:00 前广度未改善 → 不加仓
> 14:00 后龙头炸板 → 减仓
