# 涨停复盘报告 + 百度资讯整合

**日期**: 2026-03-23

## 新增: zt_review.py — 涨停复盘报告

### 报告内容
1. **涨停概览** — 当日涨停总数
2. **连板梯队** — 按连板数排列，含人气评分
3. **首板 + 概念驱动** — 每只涨停股的概念板块/上涨逻辑
4. **5日线承接选股** — 近5天涨停 + 回踩5MA有承接 + 跌破拉回
5. **人气估值** — 五维打分 (连板30 + 换手率20 + 封单额20 + 市值15 + 量比15 = 100)

### 人气估值评级
| 评级 | 分数 | 含义 |
|------|------|------|
| S | ≥75 | 极高人气，市场龙头 |
| A | 55-74 | 高人气，强辨识度 |
| B | 35-54 | 中等人气 |
| C | <35 | 一般 |

### 数据源
- MX API `stock_screen()` — 涨停列表、连板、概念板块
- EastMoney K线 API — 5日线计算和承接判断

### 定时
- 每个交易日 **15:10** 自动生成（盘后日报之后）

### 用法
```bash
python3 zt_review.py                           # 打印到终端
python3 zt_review.py --publish-dir ./reports   # 输出到文件
python3 zt_review.py --skip-ma5               # 跳过5日线分析（快速）
```
