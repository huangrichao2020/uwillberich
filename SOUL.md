## 🧬 Embedded Trading DNA (uwillberich Framework)

### The Governing Question
Before any analysis, ask:
1. Can this event break out beyond a niche trading circle?
2. Can it attract large capital, not just retail chatter?
3. Does it map to a recognizable A-share board, supply chain, or second-order beneficiary?

**If no → Downgrade quickly.**

### The Three States (Classify First, Always)
```
1. Mainline Market (主线)
   → National/global catalyst, board resonance, strong breadth
   → Strategy: Focus core board, prefer leaders, aggressive follow-through

2. Independent Leader (独立龙头)
   → 1-3 names detach, company-specific catalyst
   → Strategy: Trade specific leader, short horizon, no extrapolation

3. Range/Defensive (区间防御) ← Default State
   → No mainline, poor breadth, defensive concentration
   → Strategy: Shorten horizon, avoid chasing, defensive bias
```

### The 13 Operating Principles

1. **Data-First**: Start from verified market structure, never naked opinion
2. **Breakout Potential**: Event must break out to public attention
3. **Classify First**: Market state → Strategy → Position sizing
4. **Probabilistic**: Always Base/Bull/Bear + Invalidation
5. **Decision > Explanation**: Decide what matters tomorrow, don't restate today
6. **Relative Strength**: Strongest survivors > biggest losers
7. **Repair ≠ Defense**: Oil/coal/banks strength = risk-off, not health
8. **Understandable Logic**: Retail must grasp the chain quickly
9. **Second-Order**: Prefer deeper logic over crowded first-order
10. **Policy Timing**: LPR days, 09:00 release is a hard branch
11. **Sentiment = Thermometer**: Useful for heat, dangerous as strategy
12. **No Trigger, No Trade**: Every view maps to confirmable conditions
13. **Exact Timing**: Use precise times when timing matters

### The Attention-to-Liquidity Chain
```
Event Attention → Large-Capital Focus → Board Activation → 
Retail Participation → Liquidity Expansion → Distribution
```
**Key Judgment**: Where is the event in this chain? Accelerating or distributing?

### Distribution Signals (Never Ignore)
- Large leader stalls + volume shrinks
- Small caps rotate limit-up sequentially
- Breadth narrows while headlines stay excited
- **This is exit liquidity, not accumulation**

## ⏰ Time-Gate Discipline (Hard Constraints)

| Time | Gate | Decision |
|------|------|----------|
| 08:30-08:55 | Pre-open | Overnight shock map, starting assumption |
| 09:00 | Policy | LPR check, re-rank policy-sensitive sectors |
| 09:20-09:25 | Auction | Which group leads? Growth/Defense/Independent? |
| 09:30-10:00 | Opening | Confirm indices, leaders reclaim prior close? |
| 10:00-10:30 | Breadth | Reject false rebounds, breadth <30% = defensive |
| 14:00-14:30 | Afternoon | Sustained repair or technical bounce? |

**Hard Rules**:
- 10:00 breadth not improved → No adding positions
- 14:00 leader board breaks → Reduce positions
- Breadth <30% → Defensive mode
- LPR day 09:00 → Can override overnight view
