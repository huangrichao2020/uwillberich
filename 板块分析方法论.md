## 🎯 板块分析核心框架

### 一、三层分类法

#### 第一层：市场状态判断

**主线市场** (Mainline)
```
特征：
✓ 国家级或全球性催化剂
✓ 板块共振 (5+ 只龙头共同上涨)
✓ 广度强 (上涨家数 > 3000)
✓ 多只涨停 + 真实换手

应对策略：
→ 聚焦核心板块
→ 偏好龙头和二线流动性标的
→ 允许积极跟进
```

**独立龙头市场** (Independent Leader)
```
特征：
✓ 1-3 只股票脱离大盘
✓ 板块本身疲软或混杂
✓ 公司层面催化剂 (订单/业绩/重组)
✓ 流动性集中而非扩散

应对策略：
→ 交易特定龙头
→ 保持短周期
→ 不 extrapolate 完整板块逻辑
```

**区间/防御市场** (Range/Defensive) ← **当前状态**
```
特征：
✓ 无主线、无持久龙头
✓ 广度差 (< 1500 家上涨)
✓ 强度集中在低贝塔避险板块
✓ 主力大幅流出

应对策略：
→ 缩短持仓周期
→ 避免盲目追涨
→ 偏好观察、选择性均值回归
```

```
高強度板块特征：
1. 板块指数涨幅 > 2%
2. 主力净流入 > 5 亿
3. 涨停家数 ≥ 3
4. 板块内上涨家数 > 70%
```

### Step 2: 集合竞价分析 (09:20-09:25)

1. 哪个板块集体高开？

```
竞价强度 = (高开幅度 × 2) + (竞价量比) + (未匹配买盘)

强势竞价特征：
- 板块指数高开 > 1%
- 龙头竞价量比 > 3
```

### Step 4: 广度验证 (10:00-10:30)

**广度指标**：
```
上涨家数 / 总家数：
> 70%: 强势市场 (可积极)
50-70%: 震荡市场 (选择性)
30-50%: 弱势市场 (谨慎)
< 30%: 极弱市场 (防御) ← 今日 13.7%

涨停家数 / 跌停家数：
> 5: 情绪高涨
2-5: 情绪中性
< 2: 情绪低迷
```

**今日广度分析**：
- 上涨家数：631 / 4590 = 13.7% (极弱)

## 🧠 核心洞察

### 1. 板块强度 > 指数涨跌

指数可以失真，但板块强度不会骗人：
- 指数涨 1% 但只有 500 家上涨 = 假强
- 指数跌 1% 但有 3000 家上涨 = 假弱

**今日例证**：
- 上证指数 -1.24%，但 CPO 板块 +6%+
- 创业板指 +1.30%，但上涨仅 135/1386 = 9.7%
- 结论：指数失真，结构分化极端

### 2. 主力流向 > 涨跌幅

**关键洞察**：
- 涨 5% 但主力流出 = 散户推动 (不可持续)
- 涨 2% 但主力大幅流入 = 机构建仓 (可持续)

**今日例证**：
- 新易盛 +8.32%，主力 +15.4 亿 = 真强
- 某股票 +10% 涨停，主力 -2 亿 = 游资炒作 (次日谨慎)

### 3. 广度扩散 > 单点突破

**判断标准**：
- 单一龙头涨停 = 独立行情 (观察)
- 龙头 +2 只跟风 = 板块雏形 (轻仓试)
- 龙头 +5 只跟风 + 涨停≥3 = 主线确立 (可重仓)

**今日例证**：
- CPO：新易盛 + 中际旭创 + 多只跟涨 = 主线雏形
- 全市场：631/4590 上涨 = 无广度，不可追

### 4. 二阶受益 > 一阶直觉

**思维层次**：
```
一阶思维：石油涨 → 买石油股 (拥挤)
二阶思维：石油涨 → 买油服/替代能源 (差异化)

一阶思维：AI 火 → 买 GPU (拥挤)
二阶思维：AI 火 → 买光模块/PCB/散热 (扩散)
```

**今日例证**：
- 一阶：英伟达涨 → 买算力 (正确但拥挤)
- 二阶：算力需求 → 买 CPO 光模块 (更优选择)
- 结果：CPO 涨幅 > 算力服务器

### 5. 时间门纪律 > 主观预测

**关键时间窗口**：
- 09:00 政策发布 → 可覆盖隔夜判断
- 09:25 集合竞价 → 揭示真实意图
- 09:45 第一轮回撤 → 检验承接力
- 10:00 广度确认 → 决定仓位
- 14:00 午后确认 → 决定去留

**纪律**：
- 10:00 前广度未改善 → 不加仓
- 14:00 后龙头炸板 → 减仓
- 广度 < 30% → 防御为主

## ⚠️ 常见误区

### 误区 1: 指数涨 = 可以买
**纠正**: 看广度，今日指数涨但 631/4590 上涨 = 不可追

### 误区 2: 涨停 = 强势
**纠正**: 看主力流向，涨停但主力流出 = 游资一日游

### 误区 3: 跌多了 = 可以抄底
**纠正**: 看相对强度，最弱的板块跌再多也不是抄底对象

### 误区 4: 利好 = 会涨
**纠正**: 看市场状态，防御市场中利好常被无视

### 误区 5: 龙头涨 = 板块强
**纠正**: 看广度扩散，单一龙头 ≠ 板块行情
