## 📈 三、五大核心洞察

### 1. 板块强度 > 指数涨跌

**今日例证**:
```
上证指数 -1.24%，但 CPO 板块 +6%+
创业板指 +1.30%，但上涨仅 135/1386 = 9.7%
→ 指数失真，结构分化极端
```

### 2. 主力流向 > 涨跌幅

**关键洞察**:
```
涨 5% 但主力流出 = 散户推动 (不可持续)
涨 2% 但主力大幅流入 = 机构建仓 (可持续)

今日例证:
新易盛 +8.32%，主力 +15.4 亿 = 真强
某股票涨停但主力 -2 亿 = 游资一日游
```

### 3. 广度扩散 > 单点突破

**判断标准**:
```
单一龙头涨停 = 独立行情 (观察)
龙头 +2 只跟风 = 板块雏形 (轻仓试)
龙头 +5 只跟风 + 涨停≥3 = 主线确立 (可重仓)
```

### 4. 二阶受益 > 一阶直觉

**思维层次**:
```
一阶思维：石油涨 → 买石油股 (拥挤)
二阶思维：石油涨 → 买油服/替代能源 (差异化)

一阶思维：AI 火 → 买 GPU (拥挤)
二阶思维：AI 火 → 买光模块/PCB/散热 (扩散)

今日结果：CPO 涨幅 > 算力服务器
```

### 5. 时间门纪律 > 主观预测

**硬约束**:
```
- 10:00 前广度未改善 → 不加仓
- 14:00 后龙头炸板 → 减仓
- 广度 <30% → 防御为主
- LPR 日 09:00 必须检查 → 可覆盖隔夜判断
```
